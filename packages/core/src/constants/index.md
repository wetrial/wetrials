# constants - 预定义常量

提供一套使用频率高的预定义常量

## 使用方式

```tsx | pure
import { LAYOUT_FORM_SINGLE } from '@wetrial/core/lib/constants';
```

## API

| 名称               | 描述     | 类型                      |
| ------------------ | -------- | ------------------------- |
| LAYOUT_FORM_SINGLE | 两列布局 | `(url?:string)=>string[]` |
